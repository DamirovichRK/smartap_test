# Кроссплатформенный агент сбора информации об ОС

## О проекте

Микросервисный агент для сбора информации об операционной системе. Поддерживает Windows и Linux (Debian, Ubuntu, RedOS, Astra Linux).

### Архитектура

Проект построен на принципах микросервисной архитектуры с использованием ООП:

- **Сервис логирования** - платформозависимая запись логов
- **Сервис инвентаризации** - сбор информации об ОС
- **Сервис диспетчеризации** - управление очередью задач и воркерами
- **Фабрика сервисов** - создание нужных реализаций под конкретную ОС

## Возможности

Сбор информации об ОС (название, версия, сборка, редакция)  
Поддержка Windows (чтение из реестра)  
Поддержка Linux (Debian, Ubuntu, RedOS, Astra Linux)  
Многопоточная обработка задач  
Защита от переполнения очередей  
Белый список команд  
Детальное логирование с ротацией уровней  
Диагностика прав доступа  

## Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/DamirovichRK/smartap_test.git
cd smartap_test
```

### 2. Создание виртуального окружения
Windows: 
```bash
python -m venv venv
venv\Scripts\activate
```
*nix: 
```bash
python3 -m venv venv
source venv/bin/activate
```
### 3. Создание файла с командами

```bash
echo "inventory" > commands.txt
```
### 4. Запуск
```bash
python main.py commands.txt
```
## Файл конфига
```ini
[logging]
level = info
log_path = logs

[workers]

```
